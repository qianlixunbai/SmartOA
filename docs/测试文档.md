## 4. 审批流转完整场景

### 场景一：正常审批通过

```
zhangsan 提交请假申请
  → 状态 PENDING，currentApproverId = admin（直属领导）
  → admin 审批 APPROVE
    → 状态 PENDING，currentApproverId = zongjian1（部门总监）
    → zongjian1 审批 APPROVE
      → 状态 APPROVED，流程结束
```

### 场景二：中途驳回

```
zhangsan 提交请假申请
  → admin 审批 REJECT
    → 状态 REJECTED，流程终止
```

### 场景三：撤回

```
zhangsan 提交请假申请
  → zhangsan 撤回
    → 状态 WITHDRAWN
```

### 场景四：转派后通过

```
zhangsan 提交请假申请
  → admin（直属领导）转派给 zongjian2
    → 状态 PENDING，currentApproverId = zongjian2
    → zongjian2 审批 APPROVE
      → 状态 PENDING，currentApproverId = zongjian1（部门总监）
      → zongjian1 审批 APPROVE
        → 状态 APPROVED
```

## 5. 接口测试（curl 示例）

### 5.1 登录获取 Token

```bash
curl -X POST http://localhost:8080/api/login \
  -H "Content-Type: application/json" \
  -d '{"username":"zhangsan","password":"123456"}'
```

### 5.2 提交申请

```bash
curl -X POST http://localhost:8080/api/leave/submit \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{"templateId":1,"leaveType":"年假","startDate":"2026-06-01","endDate":"2026-06-03","reason":"test"}'
```

### 5.3 审批通过

```bash
curl -X POST http://localhost:8080/api/leave/approve \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{"requestId":1,"action":"APPROVE","comment":"同意"}'
```

### 5.4 撤回

```bash
curl -X POST http://localhost:8080/api/leave/1/withdraw \
  -H "Authorization: Bearer <token>"
```
